KFC运行相关

2023-05-24

观看本教程之前务必将软件全部切换为KFC运行相关

本教程主要讲解以上部分分支的额外内容

Asphyxia_Plugins By 22vv0

0.安装到氧无核心的plugins文件夹

基本配置：
打开氧无控制面板，找到右侧SDVX栏-->Overview目，
1.进行你需要的配置勾选
2.在该目的右侧Exceed Gear Data Directory录 填写SDVX的绝对地址（例如 E:\SDVX-KFC）
注意目标位置即soundvoltex.dll所在目录
3.点击左侧Update Webui Assets录，
然后在"Update WebUI Assets (pull from game files)"找到绿色的"Update"按钮，点击后再点击弹出窗口中的"确认/OK",等待至命令行输出以下字样
"Running....
Done."

个人数据查看(部分内容支持修改)：
右侧SDVX栏-->Profiles目-->点击 你需要查看/修改的账号的 Detail
Detail 为你目前账号总览
Setting 中你可以修改个人信息，手台显示样式，副屏背景，游玩BGM，表情（雾），修改后可以点击Play进行预览
修改完毕记得点击Submit进行提交
Score 为你所有曲目的成绩显示，并且支持修改排列顺序
Course是干什么的我也不知道，请窒息
后面两个生成器（Generator）是字面意思，不会真的有人不开解锁，手动添加吧

游戏所有曲目数据查看
右侧SDVX栏--> Songs List目，同样支持按照不同的等级进行排序和搜索

解锁部分奖励
右侧SDVX栏--> Unlocking Events目，不会真的有人不开全解锁，手动登录解锁吧
